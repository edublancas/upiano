# UPiano

A Piano in your terminal.

## Screenshot

 ![](./screenshot-upiano.png)


## How to run

Install via pip:

    pip install upiano

And then run:

    upiano

Make sure your terminal window is big enough.
The wider you can make it, the more keys you'll have! 🎹 😀
